# 初始引导

## 关于库

小幻阅读支持读书 / RSS / 播客。所有的使用数据，包括阅读进度、本地缓存、订阅等信息都需要存在一个固定的地方。

如果这些信息放在 AppsFolder 里，难免失于灵活，且容易随着应用卸载而丢失，更不易于跨设备同步。

故此需要一个用户指定一个外部文件夹，用以存放应用数据，这就是 `库`。

## 1. 库选择

如果用户是第一次使用，那他可以需要创建一个空白文件夹作为 `库`，应用会往里面写数据。

如果用户已经有了一个库，那么它可以打开库文件夹，应用将会回到库记录的状态。

## 2. 初始服务类型

这是一个向用户展示应用功能的机会。

当用户新建一个空白文件夹作为新库时，由于没有数据，应用将展示三个可选项：

- 书籍 / 漫画
- RSS
- 播客

用户可以任选其一作为起始服务，选择后，此服务将作为默认启动服务。
